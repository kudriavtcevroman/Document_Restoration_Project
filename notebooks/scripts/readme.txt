Эта директория содержит вспомогательные Python-скрипты для проекта, в основном для генерации искажений изображений.

## Назначение:
- Скрипты используются в ноутбуках для создания датасетов с реалистичными искажениями (плохая печать, засветление, пикселизация).
- Они помогают симулировать дефекты документов для обучения моделей.

## Список файлов:
- effects.py: Модуль с функциями для применения искажений:
  - apply_bad_print_effect: Эффект плохой печати (полосы, пятна, шум).
  - apply_aging_effect: Эффект старения (пятна от времени).
  - apply_brightness_contrast: Эффект засветления/затемнения (изменение яркости/контраста в областях).
  - apply_pixelation: Эффект пикселизации (масштабирование с nearest neighbor).

## Инструкции:
- Импортируйте функции в Python: from effects import apply_bad_print_effect
- Параметры функций масштабированы для высокого разрешения (e.g., 1024x1448).
- Пример использования в create_augmentation_dataset.ipynb.
- Зависимости: cv2 (OpenCV), numpy, random.
